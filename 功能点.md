## 四、校验财报模块
### 4.1 影像区域功能
| 功能编号 | 功能名称 | 功能描述 |
| --- | --- | --- |
| F050 | 报表切换展示 | 提供三张报表（资产负债表、利润表、现金流量表）的切换展示，通过点击标签页形式切换 |
| F051 | 图像模式显示 | 以图像形式展示报表影像，支持以下操作：缩放功能（放大、缩小）、拖动功能（拖动查看）、旋转功能（90度旋转）、重新上传（当前影像有误时）、分页器（支持多张图片切换） |
| F052 | 影像高亮显示 | 在图像模式下，影像会高亮显示部分科目背景和序列号，便于用户定位和核对 |
| F053 | Excel模式显示 | 以Excel表格形式展示报表内容，表格包含科目和金额，**需考虑不同Excel文件列数可能不同**，支持分页器和重新上传功能 |
| F054 | 无影像模式显示 | 当无报表影像时，展示无报表背景图，提供「立即上传」按钮供用户补充上传影像 |


### 4.2 录入区域功能
| 功能编号 | 功能名称 | 功能描述 |

| F056 | 科目表格展示 | 展示报表科目表格，表头包含：序号、项目科目、本报告期（单位：元）、操作 |
| F057 | 报表类型关联 | 表格科目数据由报表类型决定（小企业财务报表、企业财务报表格式（已执行新金融准则，新收入准则和新租赁准则）2019年版、企业财务报表格式（未执行新金融准则，新收入准则和新租赁准则）2019年版等） |
| F058 | 录入项处理 | 科目类型为「录入项」时，金额字段可编辑修改 |
| F059 | 计算项处理 | 科目类型为「计算项」时，可点击查看计算关系（展示该科目与表格中一个或多个科目的计算关系） |

| F061 | 校验公式显示 | 在表格顶部显示当前报表的校验公式，用于核对数据平衡关系 |
| F062 | 识别列选择框 | 提供当前识别列选择下拉框，**内容由后端返回**，与影像中的金额列对应，选项如：年末余额、上年年末余额等 |
